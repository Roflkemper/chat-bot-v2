# V17.9.0 — STRUCTURAL GRID REALIGN

## Что исправлено
- Убран ложный переход в `PREPARE`, когда нет триггера и локальная 1h-структура против активного блока.
- Добавлен структурный анализ 1h: repeated upper/lower rejection wicks с объёмом.
- Добавлен grid context: приоритетная сторона, потенциал импульса, количество каскадных сеток 1/2/3 по порогам 1.3% / 2.2% / 2.9%.
- `ginarea` теперь ориентируется не только на active block, а на подтверждённую структурную сторону.
- Исправлен runtime-баг с `execution_side`.
- Починен `pattern_history_store.py`.
- Восстановлен `core/execution_snapshot.py` для тестов и совместимости.
- Подправлена логика `pattern_stability`.

## Что изменено по смыслу
- Если триггера нет, а структура 1h против блока, бот теперь даёт `WAIT`, а не преждевременный `PREPARE`.
- В отчёте появился отдельный блок `СТРУКТУРА 1H` и `GRID CONTEXT`.
- Это приближает систему к твоей задаче: отдельно ручная логика, отдельно контекст для сеток.
