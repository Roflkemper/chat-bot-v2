CHAT BOT VERSION 2 — V17 SMART BEHAVIOR LAYER

Что добавлено:
- V17 слой поведенческих сигналов поверх единого V16 runtime.
- Автосканирование ситуации по таймфреймам 15m/1h раз в минуту.
- Сигналы в Telegram только при изменении состояния:
  - появился сетап
  - сетап отменён
  - ситуация изменилась
- Сигнал опирается на единый V16 snapshot:
  - режим действий
  - runtime long/short
  - прогноз рынка
  - цель движения
  - импульс
  - сценарий
  - уверенность сценария
  - авто-риск
  - слом сценария

Изменения:
- core/auto_edge_alerts.py
- config.py
- VERSION.txt

Новые значения по умолчанию:
- AUTO_EDGE_ALERTS_INTERVAL_SEC=60
- AUTO_EDGE_ALERTS_COOLDOWN_SEC=180

Логика:
- бот сканирует состояние раз в минуту
- если из PAUSE переходит в рабочее grid-состояние → сигнал о появлении сетапа
- если рабочее состояние снято → сигнал об отмене
- если изменились runtime / сценарий / риск → сигнал об изменении ситуации
