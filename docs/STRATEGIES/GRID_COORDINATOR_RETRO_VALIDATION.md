## Метрики оценки

Для каждого сигнала смотрим что произошло с ценой через 60 / 120 / 240 мин:
- **TRUE** = цена сдвинулась >0.3% в ОЖИДАЕМУЮ сторону
- **FALSE** = цена сдвинулась >0.3% в ПРОТИВОПОЛОЖНУЮ сторону
- **NEUTRAL** = цена в пределах ±0.3% (флет)

Precision = TRUE / (TRUE + FALSE), без учёта нейтралов
