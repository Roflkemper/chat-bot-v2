# legacy bat files

Эти bat/cmd-файлы убраны из корня, чтобы не перегружать релиз.
Они сохранены только как служебные/исторические сценарии и не являются основным путём запуска проекта.

Боевой набор в корне:
- RUN_BOT.bat
- RUN_TESTS.bat
- INSTALL_GIT_HOOKS.bat
- MAKE_RELEASE.bat
- PUSH_RELEASE.bat
