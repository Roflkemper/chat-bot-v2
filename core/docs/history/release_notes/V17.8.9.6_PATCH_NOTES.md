# V17.8.9.6

- Починен зеркальный запуск `FLIP PREP: WATCHING` для `LONG BLOCK`.
- `WATCHING` теперь может включаться при `alignment >= 2/3` или при `RISK/DEEP + ALT > BASE`.
- Если консенсус поддерживает активный блок, базовый сценарий получает бонус вероятности.
- `LONG/LONG GRID: WORK` без активного flip-prep больше не уходит преждевременно в `REDUCE_GRID`; теперь `HOLD_GRID`.
