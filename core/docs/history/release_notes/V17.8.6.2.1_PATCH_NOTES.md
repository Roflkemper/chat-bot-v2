# V17.8.6.2.1 — STATE MACHINE + DEPTH / DISTANCE HOTFIX

## Что добавлено
- `core/state_machine_depth.py`
  - state machine: `MID_RANGE`, `SEARCH_TRIGGER`, `PRE_ACTIVATION`, `CONFIRMED`, `OVERRUN`
  - расчет `active_side`
  - расчет `depth inside block`
  - расчет `distance to active edge`
  - расчет `distance to upper/lower edge`
  - downgrade confidence при глубоком проникновении
  - блокировка входа при `OVERRUN`
- `renderers/state_machine_renderer.py`
  - короткий боевой блок для Telegram
- `tests/test_state_machine_depth.py`
  - smoke tests на базовую логику

## Логика
- зона первична
- если цена внутри активного блока, бот не должен оставаться в `PAUSE`
- глубина внутри блока должна прямо влиять на риск и разрешение входа
- расстояние до края должно всегда быть видно трейдеру

## Шкала depth
- `0–15%` → только вошли
- `15–50%` → рабочая зона
- `50–85%` → риск прошивки, confidence -1
- `85%+` → `OVERRUN`, вход блокировать

## Пример интеграции
```python
from core.state_machine_depth import RangeLevels, evaluate_state
from renderers.state_machine_renderer import render_compact_state_block

levels = RangeLevels(low=68358.10, mid=70607.55, high=72857.00)
result = evaluate_state(price=72487.90, levels=levels, early_reaction=False, confirm_count_2of3=0)
text = render_compact_state_block("BTC", "1h", 72487.90, result)
print(text)
```
